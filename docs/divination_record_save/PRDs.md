# 梅花易数 - 起卦记录保存功能 PRD

## 1. 概述

### 1.1 功能背景

梅花易数模块已实现多种起卦方式（时空、报数、文字、手动），但缺少起卦记录的持久化保存功能。用户起卦后无法查看历史记录，也无法对起卦结果进行回顾和分析。

### 1.2 功能目标

1. **起卦记录保存**：将起卦结果保存到本地数据库
2. **历史记录查看**：提供历史记录列表，支持实时更新
3. **卜问内容整合**：在起卦时记录用户的卜问内容
4. **四柱信息展示**：在结果页面展示四柱信息（预留接口）
5. **物候信息展示**：在结果页面展示物候信息（预留接口）

### 1.3 核心价值

- **数据持久化**：起卦记录不再丢失
- **历史追溯**：用户可以回顾过去的起卦记录
- **用户体验提升**：整合卜问输入，提供完整的起卦流程

## 2. 功能需求

### 2.1 卜问输入功能

**功能描述**：在起卦前，用户可以输入卜问内容

**输入字段**：
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 占测人昵称 | 文本 | 否 | 最多12字 |
| 性别 | 选择 | 否 | 男/女/未知 |
| 生年干支 | 选择 | 否 | 支持五虎遁 |
| 占测问题 | 文本 | 是 | 最多24字 |
| 详细描述 | 文本 | 否 | 最多240字，可展开 |

**UI组件**：使用 `DivinationQuestionWidget`（来自 xuan-common）

### 2.2 起卦页面重构

**页面布局**：
```
┌─────────────────────────────────────┐
│  [卜问输入区域]                      │
│  DivinationQuestionWidget           │
├─────────────────────────────────────┤
│  [起卦方案选择]                      │
│  TabBar: 时空 | 报数 | 文字 | 手动  │
│  TabBarView: 各起卦方式的输入界面    │
├─────────────────────────────────────┤
│  [查看结果按钮]                      │
└─────────────────────────────────────┘
```

**交互流程**：
1. 用户输入卜问内容（可选）
2. 用户选择起卦方式并起卦
3. 点击"查看结果"按钮跳转到结果页面

### 2.3 结果展示页面

**页面布局**：
```
┌─────────────────────────────────────┐
│  [卜问内容卡片]（如有）              │
├─────────────────────────────────────┤
│  [四柱卡片]（预留接口）              │
│  FourZhuEightCharsCard              │
├─────────────────────────────────────┤
│  [物候信息卡片]（预留接口）          │
│  JieQiRiseSetCard                   │
├─────────────────────────────────────┤
│  [卦象展示]                          │
│  GuaDisplayWidget                   │
├─────────────────────────────────────┤
│  [起卦算法]（可折叠）                │
│  ExpansionTile                      │
├─────────────────────────────────────┤
│  [返回]  [保存]                      │
└─────────────────────────────────────┘
```

**功能特性**：
- 展示卜问内容
- 展示卦象信息（本卦、变卦、互卦）
- 展示起卦算法详情（可折叠）
- 保存起卦记录到本地数据库

### 2.4 历史记录功能

**页面布局**：
```
┌─────────────────────────────────────┐
│  标题栏：起卦历史记录                │
├─────────────────────────────────────┤
│  记录列表                            │
│  - 卡片式展示                        │
│  - 显示卜问内容                      │
│  - 显示卦象摘要                      │
│  - 显示起卦时间                      │
│  - 显示起卦方法                      │
└─────────────────────────────────────┘
```

**功能特性**：
- 实时数据流监控（Stream）
- 相对时间显示（刚刚、x分钟前、x小时前等）
- 空状态提示
- 点击查看详情（待实现）

## 3. 技术需求

### 3.1 数据库设计

**表名**：`t_meihua_gua_info`

| 字段 | 类型 | 说明 |
|------|------|------|
| uuid | TEXT | 主键 |
| divination_uuid | TEXT | 占卜记录UUID |
| question | TEXT | 卜问内容（可选） |
| original_upper_gua | INT | 本卦上卦数字 |
| original_lower_gua | INT | 本卦下卦数字 |
| changing_yao | INT | 动爻（1-6） |
| changed_upper_gua | INT | 变卦上卦数字 |
| changed_lower_gua | INT | 变卦下卦数字 |
| hu_upper_gua | INT | 互卦上卦数字 |
| hu_lower_gua | INT | 互卦下卦数字 |
| method | TEXT | 起卦方法 |
| params_json | TEXT | 起卦参数JSON |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |
| deleted_at | DATETIME | 删除时间（软删除） |

### 3.2 架构设计

```
┌─────────────────────────────────────────┐
│              UI Layer                    │
│  ├── MeiHuaDivinationPage               │
│  ├── DivinationResultPage               │
│  └── MeiHuaHistoryPage                  │
├─────────────────────────────────────────┤
│            Service Layer                 │
│  └── DivinationRecordService            │
├─────────────────────────────────────────┤
│          Repository Layer                │
│  └── MeiHuaDivinationRepository         │
├─────────────────────────────────────────┤
│              DAO Layer                   │
│  └── MeiHuaDivinationsDao               │
├─────────────────────────────────────────┤
│            Database Layer                │
│  └── MeiHuaDatabase (Drift)             │
└─────────────────────────────────────────┘
```

### 3.3 依赖关系

**外部依赖**：
- `common`：卜问输入组件、四柱卡片、物候卡片
- `persistence_core`：数据持久化核心接口
- `drift`：数据库ORM

# 文字起卦增强方案 - 实现计划

## 一、起卦方案设计

### 1.1 按字数起卦（平分字数法）
**适用场景**：长文本、成语、多字输入

| 条件 | 上卦 | 下卦 | 动爻 |
|------|------|------|------|
| 偶数字数 | 前半字数之和 % 8 | 后半字数之和 % 8 | 总字数 % 6 |
| 奇数字数 | (总字数-1)/2 字数 % 8 | (总字数+1)/2 字数 % 8 | 总字数 % 6 |

**示例**：输入"天行健君子以自强不息"（9字）
- 上卦：前4字 → 4 % 8 = 4（震）
- 下卦：后5字 → 5 % 8 = 5（巽）
- 动爻：9 % 6 = 3

---

### 1.2 按笔画起卦（精细方案）
**适用场景**：1-10个字的短文本

#### 单字起卦（拆字法）
| 字形结构 | 上卦 | 下卦 |
|---------|------|------|
| 左右结构 | 左边笔画 % 8 | 右边笔画 % 8 |
| 上下结构 | 上部笔画 % 8 | 下部笔画 % 8 |
| 包围结构 | 外部笔画 % 8 | 内部笔画 % 8 |
| 独体字 | 总笔画 % 8 | 时辰数 % 8 |

#### 多字笔画法
- 上卦：前半字笔画总和 % 8
- 下卦：后半字笔画总和 % 8
- 动爻：总笔画 % 6

---

### 1.3 按现代注音起卦（四声法）
**适用场景**：根据普通话四声

| 四声 | 数值 | 示例 |
|------|------|------|
| 一声（阴平） | 1 | 妈 mā |
| 二声（阳平） | 2 | 麻 má |
| 三声（上声） | 3 | 马 mǎ |
| 四声（去声） | 4 | 骂 mà |

**起卦规则**：
- 上卦：前半字四声值之和 % 8
- 下卦：后半字四声值之和 % 8
- 动爻：全部四声值之和 % 6

---

### 1.4 按古代注音起卦（平仄法）
**适用场景**：古诗词、文言文

| 声调 | 数值 |
|------|------|
| 平声（1、2声） | 1 |
| 仄声（3、4声） | 2 |

**起卦规则**：
- 上卦：前半字平仄值之和 % 8
- 下卦：后半字平仄值之和 % 8
- 动爻：全部平仄值之和 % 6

---

## 二、数据模型

### 2.1 文字起卦方法枚举
```dart
enum TextDivinationMethod {
  byCharCount,    // 按字数
  byStroke,       // 按笔画
  byModernTone,   // 按现代四声
  byAncientTone,  // 按古代平仄
}
```

### 2.2 字符分析结果
```dart
class CharacterAnalysis {
  final String character;
  final int strokeCount;        // 笔画数
  final int modernTone;         // 现代四声 (1-4)
  final bool isPing;            // 是否平声
  final String? structure;      // 字形结构（左右/上下/包围/独体）
  final int? leftPartStroke;    // 左部笔画
  final int? rightPartStroke;   // 右部笔画
}
```

## 七、注意事项

1. **笔画数据**：需要完整的汉字笔画数据库
2. **拼音数据**：需要准确的拼音和声调数据
3. **字形结构**：需要汉字结构分类数据
4. **性能**：大量文字分析需要异步处理
5. **降级方案**：数据缺失时的默认处理
